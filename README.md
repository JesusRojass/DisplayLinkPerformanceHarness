# DisplayLinkPerfHarness

SwiftUI app for profiling CADisplayLink and FirebasePerformance.

## Package pin
- Repo: https://github.com/JesusRojass/firebase-ios-sdk.git
- Branch: JesusRojass/#10220

## Run
1. File - Packages - Resolve Package Versions
2. Select a signing team for the app target.
3. Drop a `GoogleService-Info.plist` 
4. Run on a ProMotion device to test 120 Hz.
